Para executar o arquivo, use o terminal

Para o modo padrão: (Pixel = 1, Ciclos = 10, N° de Formigas = 5, Versão = -v1, Labirinto = M1.txt, Evaporação = 3, Taxa de Feromonio = 10
py colonia.py

O algoritmo aceita os seguinte argumentos:

py colonia.py NOMEDOLABIRINTO.txt PIXELS(int) CICLOS(int) FORMIGAS(int) VERSÃO TAXAEVAPORAÇÃO TAXAFEROMONIO

São as versões:

-v1 : Mostra somente o grau de feromonio no labirinto variando de 0 a 255 (Cor vermelha)
-v2 : Extremamente lento porém mostra a momimentação da formiga

obs¹: Será exibido um breve relatório MÉDIO no terminal ao final da execução
obs²: Observar no terminal quando termina a execução, para sair deverá apertar o "x" da janela
